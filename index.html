Mengenal HTML, CSS, JavaScript & Tailwind

HTML (HyperText Markup Language) digunakan untuk membuat struktur halaman web.

CSS (Cascading Style Sheets) berfungsi untuk mempercantik tampilan dengan warna, layout, dan efek.

JavaScript menambahkan interaktivitas seperti tombol, animasi, dan fungsi dinamis.

Tailwind CSS adalah framework CSS modern berbasis utility-class yang mempercepat proses desain web.

Tata Cara Membuat Website Sederhana

  1. Buka editor seperti VS Code.
  2. Buat file index.html, style.css, dan script.js.
  3. Tulis struktur dasar HTML.
  4. Hubungkan CSS dan JS ke HTML menggunakan tag <link> dan <script>.
  5. Simpan file, lalu buka di browser untuk melihat hasilnya.
